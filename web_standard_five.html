Standard 5.1 - Use Multimedia on the Web

Web pages are using multimedia more and more frequently to help users that access the page. Web pages consist of links, text, images, and HTML5 now even supports videos or audio files to be played directly from the web page. It is important for site creators to use multimedia for the development of their websites.

Standard 5.2 - Demonstrate Brand/Marketing/Traffic Analysis

The design of a page heavily affects the demographic of people that use the web page. My VRBO site, for example, has bright colors to attract users and compliment the fact that it is a website built for a coastal vacation resort.

Standard 5.3 - Understand the Relationship Between Web and Social Media

Social Media is an increasingly important aspect of how users spend their time on the Web. Web pages that integrate social media outlets or widgets can allow their users to connect to their site through social media or vice versa.
