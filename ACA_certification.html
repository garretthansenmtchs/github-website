Standard 1.2 - Use Styles to Format Webpages

HTML is the markup language used for creating elements on a web page, while CSS is the language used to design a web page. CSS styling can be applied to the page directly within the HTML document using the style tag, or externally using a stylesheet link tag in the HTML document. CSS can change the color, font style, and size of elements on a web page. CSS ids can change the look of a single element, while classes can be used to stylize several elements at the same time.

Standard 1.3 - Understanding Advanced CSS Selectors & Properties

HTML is comprised of structural elements, such as header, body, main, aside, and footer. These tags are used to create a layout for a web page, so that each section of a page can be individually identified using CSS. The div tag also allows a person to create their own structural elements. A div tag is essentially an empty tag with an adjustable width and height that other tags or information can belong in. Each html element has three properties which affect how it displays on the page next to other elements, the margin, padding, and border properties. My Apple Website is a good example of a page that uses those three elements to help design the look of a page. Margin is the space an element has between other objects, padding is the space an element has between its content and the border, and an element's border is what defines the boundary of the element.
