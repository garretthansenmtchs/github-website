Standard 2.1 - Demonstrate Understanding of Color Theory

Color theory is about how colors work together to create images that look appealing to viewers. A logo that I drew using HTML5 canvas uses analogous colors to create a nice looking image.

Standard 2.2 - Enhance Web Pages with Lists and Images

Images can be used to improve the quality of a website by adding a nice visual touch to the website. Lists are very commonly used to design navigation menus, which every website with more than one page will need.

Standard 2.3 - Demonstrate Understanding of the Box Model

The Box Model consists of the margin, padding, and border size of all objects on a web page. My Apple Mimic Site is a good example of how margin, padding, and border are used to design a web page, as almost every element takes advantage of those properties.

Standard 2.4 - Demonstrate Ability to Design Web Pages Using CSS

CSS is used to design web pages. Each of my websites uses CSS to shape the design of the page. CSS can be used to stylize text font, element size, page color, stylize links, text color, and much more.
