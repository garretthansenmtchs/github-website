Standard 3.1 - Appropriately Use Tables

Although none of my websites use tables, I am knowledgeable in the subject. The table tag is used to create and define a table, the colspan and rowspan attributes define the dimensions of the table, th is used to define headers of the table, and the td tag is what defines data in each cell.

Standard 3.2 - Demonstrate the Ability to Use Web Forms

My VRBO site has a functioning form, complete with buttons, input fields, and an email link. Forms are used to allow a user to communicate with the site creator so that issues may be resolved or questions may be answered.
